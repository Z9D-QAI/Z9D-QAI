# Z9D.QAI - Scientific AI Research

Welcome to the Z9D•QAI - Z9D-QAI - Z9D.QAI project.  
This is an independent research initiative focused on building high‑quality scientific AI systems, including:

- scientific text understanding  
- knowledge extraction  
- large-scale dataset engineering  
- reasoning models  
- PDF and LaTeX parsing  
- retrieval and embedding pipelines  

## Research Focus
Z9D-QAI develops tools and pipelines for processing large scientific corpora such as S2ORC, arXiv metadata, and domain‑specific datasets.  
The goal is to create a robust scientific reasoning model capable of answering complex questions, analyzing research papers, and supporting scientific workflows.

## Current Work
- dataset engineering (metadata, abstracts, fulltext)
- scientific text normalization
- PDF → structured JSON pipelines
- model training and evaluation
- retrieval-augmented generation (RAG) for scientific domains
